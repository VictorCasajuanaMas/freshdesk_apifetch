## apiFetch

app para freshdesk que realiza peticiones standard a un API para mostrar en el widget el contenido devuelto en texto plano o HTML

## ejecutar programa
fdk run

## subir a producción
Hay que probar el programa conn todas las posibles opciones de configuración para que entre en todos los if's, case y ejecute todas las funciones
generar el .zip para subir a producción: fdk pack
